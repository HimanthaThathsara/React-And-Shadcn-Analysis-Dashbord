##  Overview

<code>A responsive analytics dashboard built with React and shadcn/ui. This project demonstrates a modern UI composition using React, Tailwind CSS, and the shadcn component library for building an extensible analytics/admin dashboard.</code>

---

##  Features
- Clean, accessible UI components with shadcn/ui
- Responsive layout and theming (dark/light)
- Example charts and data visualizations
- Easy local development and deployment

### Tech stack
- React (v18+)
- Tailwind CSS
- shadcn/ui
- Charting library (e.g., Chart.js, Recharts)

##  Getting Started

###  Installation

Install React-And-Shadcn-Analysis-Dashbord using one of the following methods:

**Build from source:**

1. Clone the React-And-Shadcn-Analysis-Dashbord repository:
```sh
❯ git clone https://github.com/HimanthaThathsara/React-And-Shadcn-Analysis-Dashbord
```

2. Navigate to the project directory:
```sh
❯ cd React-And-Shadcn-Analysis-Dashbord
```

3. Install the project dependencies **Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm install
```

4. Run using the following command **Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm run dev
```
